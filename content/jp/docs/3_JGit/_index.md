{{% pageinfo %}}
作成したプロジェクトで[JGit](https://www.eclipse.org/jgit/)を使用するためのセットアップをします。
{{% /pageinfo %}}

このプログラムでは、Git操作にオープンソースライブラリの[JGit](https://www.eclipse.org/jgit/)を使用するので、そのセットアップを行います。

JGitはPure Javaで実装されたGitライブラリで、Eclipseのサイトで公開されています。
